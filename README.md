# 📊 Dashboard Logístico de Fretes
Dashboard desenvolvido em Power BI para análise de custos logísticos, com foco em fretes, volumes expedidos e desempenho por transportadora e região.

# 🚀 Objetivo
Analisar e monitorar os principais indicadores logísticos, como:
- Valor total gasto com fretes
- Volume de envios
- Peso total transportado
- Transportadoras mais utilizadas
- Regiões com maior custo de frete

# 📈 Principais Insights
- Identificação das transportadoras com maior custo
- Análise de gastos ao longo dos meses
- Comparação de custo médio por estado e cidade

## 🛠️ Ferramentas utilizadas
- Power BI
- Excel / CSV (base de dados)
- DAX para cálculos e métricas
